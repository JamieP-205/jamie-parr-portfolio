Serverless app project

Talk With Jamie

A privacy-conscious personal chat app built on Netlify. It supports guest sessions, registered accounts, persistent conversations, admin tools, rate limiting and private context retrieval.

The main technical challenge was keeping public chat behaviour separate from private administration features. Raw source material stays out of GitHub and the deployed site, while the app uses serverless functions, signed cookies, Netlify Blobs and automated tests to keep the project controlled.

  • Netlify Functions
  • Authentication
  • Netlify Blobs
  • Context retrieval
  • Privacy boundaries
  • Automated tests
  • JavaScript
  • API routes
