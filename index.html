Playable Godot campaign

The World Forgot Us

A top-down game set around a failed British civil-warning network. What began as a small vertical slice is now a playable four-area campaign with combat, saving, upgrades, environmental puzzles, optional records and three endings.

The technical core is persistent campaign state across authored and runtime-built maps. The same Trace Receiver drives exploration, enemy shield windows and story discovery, while region-aware lighting, generated normal maps and procedural audio give each area its own identity.

Adaptive French platform

French for Life

An accuracy-first French-learning platform built around one strict rule: teach language before asking the learner to produce it. The public no-account demo runs in the browser and keeps progress locally while production account sync remains gated behind verified email delivery and privacy review.

A concept graph declares prerequisites, scored prompts are checked against a vendored FLELex vocabulary source, and deterministic tests protect the difference between finishing an activity and demonstrating learning.

Private adaptive learning system

Groundwork

A local React app built to rebuild my first-year computing knowledge and prepare for second year. It chooses the next useful activity from recorded evidence, explains why it was selected and prevents tests or portfolio work from appearing before the required teaching.

Groundwork includes a real SQLite-powered fifteen-stage SQL path, checkpoint gating, mistake-driven remediation, a personal Guidebook derived from evidence, migrations for older progress data and continuous responsive checks from 320 to 1920 pixels.

Live production website

Coast Internet Radio

The website for a small Newry radio station with real listeners. The station stream is HTTP while the site is HTTPS, so two Cloudflare Workers safely re-serve the audio and now-playing feed for modern browsers.

The project also includes an authenticated owner-admin area, play history, listener feedback, first-party analytics, CSP validation and automated authentication tests.

  • Cloudflare Workers
  • Netlify Functions
  • Netlify Blobs
  • Session auth and CSRF
  • First-party analytics
  • Content Security Policy

Privacy-conscious serverless app

Talk With Jamie

An AI chat with guest and registered accounts, saved conversations and a private admin side. The important engineering problem is access control: public and private users receive context from different retrieval boundaries enforced in code rather than left to a prompt.

Private source material is not committed or included in the static deployment. Signed sessions, storage, rate limits and automated checks sit behind Netlify Functions.

  • Authentication
  • Netlify Blobs
  • Context retrieval
  • Privacy boundaries

Local Web Fix

A clear, mobile-first service site for practical website and online-information fixes. A business can submit a public link for a free check before deciding whether paid work is worthwhile.
