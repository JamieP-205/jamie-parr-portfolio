Local adaptive learning system

Groundwork

A React app that chooses the next useful study task from recorded evidence instead of presenting another dashboard to manage.

Core problem
Deterministic guidance without pretending the system is machine learning
Proof
206 runtime assertions, 45 browser checks and 15 real SQLite stages
Data model
Versioned local progress with import, export, migration and recovery paths
  • React
  • TypeScript
  • SQL.js
  • Playwright
  • Local-first data

The repository is private because it contains my personal learning history and current course-planning material.

What made it difficult

Assessment scores cannot stand in for teaching. The recommendation rules, SQL path, portfolio gates and Guidebook all derive from the same evidence model and are tested against that boundary.
