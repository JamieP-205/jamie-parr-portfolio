# Project selection

The homepage is ordered by what each project proves, not by when it was made.

## Flagship builds

- The World Forgot Us: the widest technical range, with a playable campaign, persistent game state, authored and runtime-built areas, lighting, audio, automated checks and a browser release.
- French for Life: the strongest full-stack learning product, with curriculum rules, deterministic assessment, account safeguards and release checks.
- Groundwork: the strongest systems and testing project, with evidence-based recommendations, real SQLite work, migrations and extensive browser/runtime verification.

## Production and service work

- Coast Internet Radio: real users, unusual streaming constraints, owner administration, analytics, authentication and deployment hardening.
- Talk With Jamie: serverless application design, storage, authentication and privacy boundaries.
- Local Web Fix: clear service design, mobile usability and careful client boundaries.

Projects are described using repository evidence and their limitations remain visible. Smaller repositories and public source snapshots stay accessible through the GitHub profile rather than receiving equal space on the homepage.
