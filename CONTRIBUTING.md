# Contributing

This is a personal portfolio, so changes should remain accurate, focused, and respectful of the owner’s personal information.

1. Create a branch from `main`.
2. Run `npm install` and `npm test`.
3. Keep the site dependency-free unless a clear need is documented.
4. Test keyboard navigation, mobile layout, dark mode, large text, and reduced motion.
5. Do not add claims, qualifications, employers, contact details, or personal documents without Jamie’s approval.
